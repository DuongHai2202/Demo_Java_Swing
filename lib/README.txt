# MySQL Connector JAR

Tải MySQL Connector/J từ: https://dev.mysql.com/downloads/connector/j/

Đặt file JAR vào thư mục này.
Tên file phải bắt đầu với: mysql-connector
Ví dụ: 
  - mysql-connector-java-8.0.33.jar
  - mysql-connector-j-8.0.33.jar
  - mysql-connector-java-8.4.0.jar (bất kỳ phiên bản nào)

Script sẽ tự động tìm file JAR phù hợp!

Tải trực tiếp (phiên bản 8.0.33):
https://repo1.maven.org/maven2/com/mysql/mysql-connector-j/8.0.33/mysql-connector-j-8.0.33.jar
